# Salt Labs

Landing page hub for Visit Salt Lake internal tools and intelligence platforms.

## Apps

- **SaltLakeIQ** — Marketing intelligence platform (Live)
- **The Dam** — Digital asset management (In Development)
- **WebNotes** — Research capture tool (In Development)

## Stack

Static HTML/CSS/JS — no build step required.

## Deploy

Hosted on Railway as a static site. Push to `main` to deploy.

## Local Development

```bash
# Open directly in browser
open index.html

# Or use any static server
npx serve .
```

## Structure

```
salt-labs/
├── index.html          # Main landing page (all CSS inlined)

└── README.md
```
